## Lab 3
Три режима работы устройства защиты:

- `passthrough` - только проксирование, без противодействия.
- `jitter` - добавление случайных задержек, ограничивающее пропускную способность скрытого канала.
- `normalize` - выдача пакетов по фиксированному кванту с пакетами-заполнителями, устраняющая данный тайминговый скрытый канал.

### Запуск

Доступны alias-команды для запуска:

```bash
uz-pass
uz-jitter
uz-normalize
p2-pass
p2-jitter
p2-normalize
p1-pass
p1-jitter
p1-normalize
```
